# Machine Learning, Deep Learning & NLP Notes

A structured study archive of notebooks and implementation examples for core machine learning, deep learning, and natural language processing topics.

## What this repository contains

- Linear regression and logistic regression
- SVM, KNN, decision trees, and random forests
- PCA, clustering, anomaly detection, and boosting methods
- Practical notebook-based exercises for hands-on learning

## How it is organized

Each topic is grouped into its own folder so you can move through the concepts in a clear sequence.

## What is intentionally excluded

- Environment files such as .env
- Large data files and datasets
- Extra folders for experiments, notes, and non-essential workspace content

## Getting started

1. Open the folder for the topic you want to study.
2. Run the notebooks in a Python environment with Jupyter installed.
3. Use them as a reference while building your own models and experiments.

## Purpose

This repository serves as a personal learning record and a practical reference for understanding the foundations of modern machine learning.
